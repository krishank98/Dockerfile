## Usage

Run:

```
$ docker run --user "$(id -u):$(id -g)" -it -v $PWD/certs:/output ruanbekker/minica --domains 192.168.0.20.nip.io
```

View:

```
$ find ./certs -type f
./certs/minica-key.pem
./certs/192.168.0.20.nip.io/key.pem
./certs/192.168.0.20.nip.io/cert.pem
./certs/minica.pem
```
